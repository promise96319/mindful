# 冥想网站设计文档

> 日期：2026-01-29

## 项目概览

一个面向所有年龄段的冥想学习网站，提供从入门到进阶的完整学习路径和丰富的练习工具。纯静态网站，无需用户登录，支持中英双语。

---

## 一、整体架构

### 项目结构

```
meditation-website/
├── src/
│   ├── components/      # 可复用组件
│   ├── pages/           # 页面组件
│   ├── tools/           # 冥想工具组件
│   ├── content/         # 文档内容（Markdown）
│   ├── i18n/            # 多语言配置
│   ├── hooks/           # 自定义 hooks
│   ├── utils/           # 工具函数
│   └── assets/          # 音频、图片等静态资源
├── public/
└── package.json
```

### 技术选型

- **React 18** + **Vite** - 快速构建和开发体验
- **Tailwind CSS** - 原子化样式
- **React Router** - 页面路由
- **react-i18next** - 国际化方案，支持语言扩展
- **localStorage** - 保存用户偏好和练习统计数据
- **Howler.js** - 音频播放（环境音、引导音频）

### 部署方式

纯静态网站，可部署到 Vercel / Netlify / GitHub Pages，零服务器成本。

## 二、页面结构与导航

### 主要页面

1. **首页** `/` - 网站介绍、快速入口、特色展示
2. **学习中心** `/learn` - 冥想文档，从基础到高级的课程体系
3. **练习工具** `/tools` - 所有互动工具的入口
4. **关于** `/about` - 网站介绍、冥想的价值

### 导航设计

顶部固定导航栏，简洁清晰：

```
[Logo: 静心]    学习中心    练习工具    关于    [语言切换: 中/EN]
```

### 学习中心子页面结构

```
/learn
  /learn/basics          # 基础篇
  /learn/breathing       # 呼吸篇
  /learn/focus           # 专注篇
  /learn/relaxation      # 放松篇
  /learn/meditation      # 冥想篇
  /learn/advanced        # 进阶篇
```

每个篇章包含多个章节，用户可以按顺序学习，也可以跳转到感兴趣的部分。侧边栏显示当前篇章的目录，方便导航。

### 练习工具子页面

```
/tools
  /tools/breathing       # 呼吸练习
  /tools/focus           # 专注训练
  /tools/relaxation      # 放松练习
  /tools/meditation      # 冥想引导
  /tools/timer           # 自由计时器
  /tools/ambient         # 环境音
```

## 四、练习工具详细设计

### 1. 呼吸练习工具 `/tools/breathing`

- 可视化动画：圆圈随呼吸节奏扩张/收缩
- 多种呼吸模式可选：
  - 腹式呼吸（吸4秒-呼6秒）
  - 4-7-8 呼吸（吸4秒-屏7秒-呼8秒）
  - 方块呼吸（吸4-屏4-呼4-屏4秒）
  - 自定义节奏
- 练习时长选择：1/3/5/10 分钟
- 可选：语音提示、背景音乐
- 完成后显示统计

### 2. 专注训练工具 `/tools/focus`

- 屏幕中心显示专注点（如蜡烛火焰动画、圆点）
- 定时器倒计时
- 可选柔和背景音
- 练习时长：1/3/5/10 分钟
- 简短的开始和结束引导语

### 3. 放松练习工具 `/tools/relaxation`

- 身体扫描引导模式：配合身体部位图示，逐步引导放松
- 渐进式肌肉放松模式：紧张-放松的循环引导
- 音频引导播放
- 可选背景环境音

### 4. 冥想引导工具 `/tools/meditation`

- 多个预设冥想音频（5/10/15/20 分钟）
- 正念冥想、慈悲冥想等不同主题
- 播放进度显示
- 可叠加环境音

### 5. 自由计时器 `/tools/timer`

- 简洁的圆形计时器界面
- 预设时长快捷选择：5/10/15/20/30 分钟
- 支持自定义时长
- 可选功能：
  - 开始/结束铃声（多种风格：钟声、木鱼、颂钵）
  - 间隔提醒（每隔 N 分钟轻响一次）
  - 背景环境音
- 完成后自动记录到本地统计

### 6. 环境音工具 `/tools/ambient`

- 多种自然音效可混合播放：
  - 雨声、雷雨、细雨
  - 海浪、溪流、瀑布
  - 森林、鸟鸣、虫鸣
  - 篝火、风声
  - 白噪音、粉噪音
- 每种音效独立音量控制
- 支持同时播放 2-3 种音效混合
- 可设置播放时长或持续播放
- 可在其他工具中作为背景使用

### 7. 练习统计（本地存储）

- 记录每次练习：日期、工具类型、时长
- 首页或单独统计页展示：
  - 今日/本周/本月练习总时长
  - 连续练习天数
  - 简单的日历热力图
- 数据存储在 localStorage，隐私安全

## 五、视觉设计与UI风格

### 色彩方案

参考潮汐 App 的清新自然风格，以蓝色系为主：

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F7F8FA` | 雾白 |
| 次背景 | `#EEF1F5` | 浅灰蓝 |
| 卡片背景 | `#FFFFFF` | 纯白 |
| 主文字 | `#2C3E50` | 深蓝灰 |
| 次文字 | `#7F8C9A` | 灰蓝 |
| 主强调色 | `#5B8FB9` | 静谧蓝（按钮、进度、重点）|
| 辅助色 | `#7FB9A2` | 雾绿（自然感、成功状态）|
| 深色点缀 | `#34495E` | 深青（图标、标题）|
| 浅色点缀 | `#A8D4E6` | 天蓝（背景装饰、hover 状态）|

整体感觉：如清晨的湖面，宁静、清新、自然，让人感到平和与放松。

### 字体选择

- 中文：思源黑体（标题加粗 + 正文常规）
- 英文：Inter（统一使用，简洁现代）

### UI 特点

- 大量留白，呼吸感强
- 圆润的卡片和按钮（border-radius: 12-16px）
- 柔和的阴影，无硬边
- 动画缓动使用 ease-out，节奏舒缓
- 工具页面采用沉浸式全屏设计，减少干扰

### 响应式设计

- 移动端优先，支持手机、平板、桌面
- 导航在移动端折叠为汉堡菜单

## 六、国际化架构

### i18n 技术方案

使用 `react-i18next`，结构清晰易扩展：

```
src/i18n/
├── index.ts              # i18n 初始化配置
├── locales/
│   ├── zh-CN/
│   │   ├── common.json   # 通用文案（导航、按钮等）
│   │   ├── home.json     # 首页
│   │   ├── learn.json    # 学习中心
│   │   └── tools.json    # 练习工具
│   └── en/
│       ├── common.json
│       ├── home.json
│       ├── learn.json
│       └── tools.json
```

### 文档内容的多语言处理

学习中心的长文档采用 Markdown 文件：

```
src/content/
├── zh-CN/
│   ├── basics/
│   │   ├── 01-what-is-meditation.md
│   │   ├── 02-benefits.md
│   │   └── ...
│   └── breathing/
│       └── ...
└── en/
    ├── basics/
    └── ...
```

### 语言切换

- 顶部导航右侧语言切换按钮
- 用户选择保存到 localStorage
- 首次访问根据浏览器语言自动检测
- URL 不变（语言不体现在路径中），保持简洁

### 扩展新语言

只需添加新的语言文件夹（如 `ja`、`ko`），并在配置中注册即可。

## 七、音频资源与数据管理

### 音频资源

```
src/assets/audio/
├── ambient/           # 环境音（循环播放）
│   ├── rain.mp3
│   ├── ocean.mp3
│   ├── forest.mp3
│   ├── stream.mp3
│   ├── fire.mp3
│   ├── wind.mp3
│   ├── birds.mp3
│   └── white-noise.mp3
├── bells/             # 铃声（开始/结束提示）
│   ├── bowl.mp3       # 颂钵
│   ├── bell.mp3       # 钟声
│   └── chime.mp3      # 风铃
└── guides/            # 引导音频
    ├── zh-CN/
    │   ├── meditation-5min.mp3
    │   ├── meditation-10min.mp3
    │   ├── body-scan.mp3
    │   └── ...
    └── en/
        └── ...
```

### 音频来源建议

- 环境音：使用免费商用授权的音效（如 Freesound.org、Pixabay）
- 引导音频：后期可录制或使用 TTS 生成

### 本地数据存储

使用 localStorage 存储：

```typescript
// 用户偏好
preferences: {
  language: 'zh-CN',
  defaultBell: 'bowl',
  defaultDuration: 10
}

// 练习记录
practiceHistory: [
  { date: '2024-01-15', tool: 'breathing', duration: 300 },
  { date: '2024-01-15', tool: 'meditation', duration: 600 },
  ...
]
```

封装 `useLocalStorage` hook 统一管理读写。
