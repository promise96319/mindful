# 冥想网站设计文档

> 日期：2026-01-29
> 状态：已实现

## 项目概览

一个面向所有年龄段的冥想学习网站，提供从入门到进阶的完整学习路径和丰富的练习工具。纯静态网站，无需用户登录，支持中英双语。

## 技术栈

- React 18 + Vite
- Tailwind CSS
- react-router-dom
- react-i18next
- Howler.js（音频播放）
- localStorage（本地存储）

## 已实现功能

### 页面

- 首页：网站介绍、快速入口、特色展示、统计卡片
- 学习中心：6大篇章的课程体系
- 练习工具：6种练习工具
- 关于页面

### 练习工具

1. 呼吸练习 - 支持腹式呼吸、4-7-8、方块呼吸
2. 专注训练 - 视觉焦点训练
3. 放松练习 - 身体扫描引导
4. 冥想引导 - 正念/慈悲/睡前冥想
5. 冥想计时器 - 自定义时长
6. 环境音 - 多种自然声音混合

### 学习内容

- 基础篇：5篇完整文章
- 呼吸篇：5篇完整文章
- 专注篇：2篇核心文章
- 放松篇：1篇核心文章
- 冥想篇：2篇核心文章
- 进阶篇：1篇核心文章

### 其他功能

- 中英双语支持
- 练习统计（本地存储）
- 响应式设计

## 视觉风格

参考潮汐 App 的清新自然风格，以蓝色系为主：

- 主强调色：#5B8FB9（静谧蓝）
- 辅助色：#7FB9A2（雾绿）
- 背景色：#F7F8FA（雾白）

## 运行方式

```bash
npm install
npm run dev
```

## 部署

纯静态网站，可部署到 Vercel / Netlify / GitHub Pages。
