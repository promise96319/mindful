# 你的第一次冥想

现在，让我们进行一次简单的5分钟冥想。这是一个基础的呼吸冥想，适合完全的初学者。

## 准备就绪

找到一个舒适的坐姿，可以是椅子上或地板上。确保你不会被打扰几分钟。

## 开始冥想

### 第1分钟：安顿身心

1. 轻轻闭上眼睛
2. 做三次深呼吸，每次呼气时让身体更加放松
3. 让呼吸恢复自然的节奏
4. 感受身体与座位的接触，感受重力

### 第2-3分钟：关注呼吸

1. 将注意力带到呼吸上
2. 注意吸气时腹部或胸腔的起伏
3. 注意呼气时的感觉
4. 不需要改变呼吸，只是观察它

**当思绪飘走时**

这是完全正常的。当你发现自己在想其他事情时：

- 不要批评自己
- 轻轻地注意到"我在想事情"
- 温和地将注意力带回呼吸
- 这个"带回来"的过程就是冥想的核心练习

### 第4分钟：扩展觉察

1. 保持对呼吸的觉察
2. 同时注意身体的其他感觉
3. 可能是手放在腿上的感觉
4. 可能是空气接触皮肤的感觉

### 第5分钟：结束

1. 慢慢将注意力扩展到整个身体
2. 注意周围的声音
3. 轻轻动动手指和脚趾
4. 当你准备好时，慢慢睁开眼睛

## 冥想后

花几秒钟感受一下现在的状态。你可能感到：

- 更加平静
- 更加清醒
- 或者没有特别的感觉

这些都是正常的。冥想的效果往往是累积的，随着持续练习会逐渐显现。

## 常见问题

**"我一直在想事情，是不是做错了？"**

没有。思绪游离是正常的。每次你注意到并将注意力带回来，你就在成功地练习冥想。

**"我感觉很无聊/烦躁"**

这也是正常的。尝试对这种感觉保持好奇，观察它在身体的哪个部位，它是什么样的感觉。

**"我差点睡着了"**

如果你很困，可能需要选择一个更清醒的时间练习，或者尝试睁开眼睛、挺直身体。

## 下一步

恭喜你完成了第一次冥想！建议：

- 明天再试一次
- 保持每天5分钟的练习
- 当你感到舒适时，可以逐渐增加时间
- 使用我们的呼吸练习工具来辅助练习

记住，每一次练习都有价值，无论你觉得"做得好不好"。关键是坚持。
